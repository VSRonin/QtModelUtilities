Detailed Description

This proxy inverts the rows and columns of the source model.

This model will make the rows of the source model become columns of and vice-versa If the model is a tree the parents will be transposed as well. For example, if an index in the source model had parent index(2,0) it will have parent index(0,2) in the proxy
